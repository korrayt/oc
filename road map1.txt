Sen core-I projesini devralıyorsun.

PROJE ADI:
core-I

GENEL AMAÇ:
core-I, Windows masaüstünde çalışan, local-first / offline-first tasarlanmış, Ollama üzerinden yerel LLM kullanan, Tauri + React + TypeScript tabanlı modüler AI çekirdeğidir.

core-I sonradan .corepack paketleriyle yeni yetenekler, roller, personlar, uzmanlıklar, veri setleri, iş akışları ve araçlar kazanabilecek.

ŞU ANKİ DURUM:
Ubuntu geliştirme ortamı çöktü. Bu yüzden geliştirme Windows’a taşınıyor.

Elimizdeki klasör büyük ihtimalle tam Tauri projesi değil; önceki core-I v0.1-alpha frontend dosyalarını içeren bir paket olabilir.

Eğer klasörde src-tauri yoksa:
Bu klasörü tam Tauri projesi sanma.
Mevcut React / TypeScript frontend dosyalarını koru.
Eksik minimal Tauri kabuğunu ekle.

HEDEF PLATFORM:
Öncelik Windows.
Hedef:
npm install
npm run tauri dev

Sonra:
npm run tauri build

Final hedef:
Windows .exe / installer üretmek.

TEKNOLOJİ:
- Tauri
- React
- TypeScript

- Rust / Cargo
- Ollama
- localStorage ile geçici yerel hafıza
- İleride SQLite / local database

LİSANS MANTIĞI:
Şu an prototipte:
COREI- ile başlayan token geçerli kabul edilebilir.

Dev lisans butonu:
“Dev lisansı aktif et”

Dev token:
COREI-DEV-LOCAL-0001

Lisans durumları:
- missing
- valid
- invalid
- restricted

Lisans geçerliyse:
Chat aktif.

Lisans yoksa veya geçersizse:
Chat sınırlı modda.
Kullanıcıya açıkça bildir.

EULA, gizlilik bildirimi ve telemetry/audit rızası ayrı gösterilmeli.
Telemetry varsayılan olarak zorunlu olmamalı.
Offline konuşmalar cihazda kalmalı.
